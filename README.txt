Project Set-up Instructions:

1. To import the project into Eclipse IDE, follow these instructions.
	
	File -> Open Projects From File System -> Provide Folder Path Under 'Import Source' -> Finish

2. The Project is now Set up in the Eclipse. To run the application, right click on the project folder and Click "Run As" -> Java Application.


Input File Requirements:

1. In the attributes file, the attribute delimiter should be ' ' (i.e., each attribute should be separated by a space ' ' character)
2. In the data file, the delimiter should be ',' (i.e., the data in each row should be separated by a comma ',' character)
